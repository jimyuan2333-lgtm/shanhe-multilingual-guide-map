# 山河古镇多语智慧导览地图

Shanhe Ancient Town Multilingual Guide Map is a React-based interactive demo for a cultural tourism guide platform.

## Features

- Zoomable and draggable scenic map
- POI category filters and keyword search
- Chinese and English interface switching
- Recommended routes and smart route simulation
- POI detail panel, public facilities, current location, legend, and QR guide mockup
- Static build for Vercel deployment

## Run Locally

```bash
npm install
npm run dev
```

Open `http://localhost:5173`.

## Build

```bash
npm run build
```

The static site is generated in `dist/`.
